#Карманный прораб

> Управление строительными проектами в кармане. Сметы, расходы, фотографии, PDF-документы.

##О проекте

Веб-приложение для ИП и небольших строительных бригад. Позволяет вести проекты, составлять сметы, отслеживать расходы и генерировать красивые PDF-документы для клиентов.

###Основные возможности

- Управление проектами (статусы, сроки, заказчики);
- Смета работ с автоматическим подсчётом;
- Расходы и список покупок;
- Категории и шаблоны работ (продукты);
- Генерация PDF-сметы для клиента;
- Адаптивный дизайн (работает на телефоне).

##Быстрый старт

###Локальный запуск

```bash
#Клонируйте репозиторий
git clone https://github.com/ваш-аккаунт/karman_prorab.git
cd karman_prorab

#Создайте виртуальное окружение
python -m venv venv

#Активируйте его

#Windows:
venv\Scripts\activate
#Mac/Linux:
source venv/bin/activate

#Установите зависимости
pip install -r requirements.txt

#Запустите приложение
python backend/app.py

После запуска откройте в браузере: http://127.0.0.1:5000

###Настройка компании
 
1. Перейдите в раздел «Настройки» (кнопка ⚙️ на главной);
2. Введите данные вашей компании (название, телефон, email);
3. Эти данные автоматически подставятся во все PDF-сметы.

###Структура проекта:

###Технологии 

1. Flask — веб-фреймворк
2. SQLAlchemy — ORM для работы с БД
3. SQLite — база данных (поддерживается PostgreSQL)
4. WeasyPrint — генерация PDF
5. Bootstrap 5 — интерфейс

###PDF-смета

Приложение умеет генерировать красивое коммерческое предложение в PDF:

1. Шапка с данными компании (из настроек)
2. Информация об объекте и заказчике
3. Таблица работ с количеством, ценой и итогом
4. Блок для подписей
